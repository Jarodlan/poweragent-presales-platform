# 项目文档总览

这里集中存放本项目的产品、技术、开发、模板与演示材料，便于后续维护和查找。

当前项目已形成两条主业务线：

1. `电力行业解决方案生成智能体（0.1.0-mvp）`
2. `客户需求分析智能体（0.1.0-mvp）`

并且已经具备两者之间的联动方向：

- 客户需求分析报告
- 人工确认与编辑
- 一键带入解决方案生成工作台

## 目录结构

- `01-产品设计/`
  - PRD、页面原型说明、Demo 实施方案等
- `02-技术设计/`
  - 技术架构、接口、数据库、Prompt、工作流、场景路由、账户权限体系等
- `03-开发计划/`
  - 前后端任务拆解、PostgreSQL 本地开发说明等
- `04-知识库与模板/`
  - 解决方案模板、元数据字典、分类设计表、参考模板内容等
- `05-演示材料/`
  - PPT 与配套生成脚本
- `06-原始资料/`
  - 官网 HTML 等原始参考资料
